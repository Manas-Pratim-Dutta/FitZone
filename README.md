# 💪 FITZONE Gym Website

A modern, responsive multi-page gym website built using **Node.js, Express, HTML, Tailwind CSS, and Vanilla JavaScript**.  
This project is designed as a frontend + backend learning project with real-world structure.

---

## 🚀 Features

- 🏠 Home page with modern hero section
- 🏋️ Training Programs page
- 🎯 Online Coaching page
- 📞 Contact page
- 🎨 Modern UI built with Tailwind CSS
- ⚡ Fast Node.js + Express server
- 📱 Responsive design (mobile + desktop)
- 🧭 Multi-page routing system

---

## 🧱 Tech Stack

**Frontend:**
- HTML5
- Tailwind CSS (CDN)
- JavaScript (Vanilla)

**Backend:**
- Node.js
- Express.js
